## 🎯 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/clarriu97/insert-coin.git
   cd insert-coin
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. Start playing! 🎮
